# Черновик PRD: промо-страница тарифа Pro (Q2)

## Контекст и цель
- **Цель:** поднять конверсию trial → оплата за счёт целевого лендинга кампании Q2 для МСБ (РФ/СНГ), без изменений ядра продукта.
- **Аудитория:** владельцы малого бизнеса, первый контакт из контекстной рекламы.

## Scope (входит)
- Одностраничный промо-лендинг с одним primary CTA на старт trial тарифа Pro.
- Блоки: hero, ценность Pro, таблица «что входит», социальное доказательство (по возможности), FAQ 5–7, финальный CTA.
- Локализация: русский (основной); EN — post-MVP.
- Аналитика базовых событий: показ hero, клик primary CTA, успешный старт trial / шаг checkout (без лишнего PII).

## Out of scope
- Новые платёжные методы (только согласование с compliance).
- Изменение тарифной сетки или биллинга продукта.
- Полноценный A/B оффера в первой итерации (baseline сначала).

## Метрики успеха
- **Proposed primary:** доля trial → оплата среди квалифицированных визитов кампании.
- **Secondary:** CTR primary CTA, доля стартов trial, отказы на шаге оплаты.

## Риски и зависимости
- Нужна финализация формулировок trial/автопродления с legal.
- Согласованный источник правды по составу Pro (продукт/биллинг).
- Открытые вопросы: единый primary KPI (см. user stories и acceptance); план A/B — отдельным инкрементом.
