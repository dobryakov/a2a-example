# User stories — промо Pro (MoSCoW)

## Must have
1. **Как** посетитель с рекламы, **я хочу** сразу понять выгоду Pro и условия trial, **чтобы** начать пробный период без лишних кликов.
2. **Как** посетитель, **я хочу** видеть таблицу включённых возможностей Pro, **чтобы** сравнить с текущей потребностью.
3. **Как** посетитель, **я хочу** прочитать короткий FAQ про оплату и trial, **чтобы** снять юридические и практические сомнения.
4. **Как** команда, **я хочу** один явный primary CTA above the fold, **чтобы** не размывать конверсию.

## Should have
5. **Как** посетитель, **я хочу** видеть доверие (логотипы, цифры, отзывы), **чтобы** снизить скепсис при холодном трафике.

## Could have
6. **Как** посетитель, **я хочу** вторичный путь с короткой лид-формой, **чтобы** оставить контакт без trial (если политика данных позволяет).

## Won't have (в этой итерации)
7. **A/B** двух версий оффера в одном релизе — отложено после baseline-метрик.
8. **Новые** способы оплаты — вне scope до отдельного решения compliance.
